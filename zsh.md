# ZSH

## install zsh

[https://github.com/robbyrussell/oh-my-zsh/wiki/Installing-ZSH](https://github.com/robbyrussell/oh-my-zsh/wiki/Installing-ZSH)

1. `brew install zsh zsh-completions`
2. `zsh --version`
3. `sudo echo "$(which zsh)" >> /etc/shells`
4. `chsh -s $(which zsh)`
5. restart terminals
