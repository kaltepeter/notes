## create react app

```bash
create-react-app my-app --typescript
```

## with lerna
```bash
lerna add @types/jest
lerna add @types/node
lerna add @types/react
lerna add @types/react-dom
lerna add react 
lerna add react-dom
lerna add react-scripts
lerna add typescript 
lerna add tslint
```
